Flex Property implementation

It is a layout used to rrange elements in 1D(row/column)

Flex container
The parent element that acts like a flixible box to its child element.
Flex item
Children of Flex-container. By default any element which is a child of flex container.
Main axis
Primary axis along which all the flex-item will be laid.
Cross axis
Axis perpendicular to the main axis
